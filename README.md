# UNDERpost.net | Network System


![underpost.net](https://underpost.net/underpost-social.jpg)


https://underpost.net


## Usage:


- install:


`npm install`


`node install`


`node update`


- config:


`node console.mjs`


- app dev mode init:


`npm run dev`


- app production mode init:


`npm run start`
